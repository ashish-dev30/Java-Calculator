# 🧮 Java Calculator

A professional Java Console Calculator built while learning Core Java.

This project demonstrates Java fundamentals including Object-Oriented Programming (OOP), Exception Handling, Arrays, File Handling, and Git/GitHub.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🧮 Modulus
- 🔢 Power
- 📈 Maximum Number
- 📉 Minimum Number
- 📊 Average
- 💯 Percentage
- ⬜ Square
- 🧊 Cube
- 🔥 Factorial
- 🔍 Prime Number Check
- √ Square Root

---

## 📜 Calculator History

- Stores calculation history during runtime.
- Saves history permanently using File Handling.
- Loads previous history after restarting the application.
