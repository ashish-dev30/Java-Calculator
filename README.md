# 🧮 Java Calculator

A professional **Java Calculator** built while learning **Core Java**.

This project started as a **Console Calculator** and has now evolved into a **Java Swing GUI Calculator** with keyboard support.

It demonstrates Java fundamentals including Object-Oriented Programming (OOP), Exception Handling, Arrays, File Handling, Java Swing GUI, Event Handling, and Git/GitHub.

* Stores calculation history during runtime.
* Saves history permanently using File Handling.
* Loads previous history after restarting the application.
